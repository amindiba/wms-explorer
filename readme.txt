سیستم جامع مدیریت مرجوعی، بازپرداخت و مبادله کالا در ووکامرس — کاملا فارسی و راستچین.

== Description ==

مدیریت مرجوعی ووکامرس امکانات زیر را فراهم می‌کند:

* **پورتال مشتری** در حساب کاربری — بدون نیاز به تماس با پشتیبانی
* **مرجوعی مهمان** از طریق شماره سفارش + ایمیل
* **شماره RMA** با شماره‌گذاری خودکار و پیشوند سفارشی
* **شرایط مرجوعی** — پنجره زمانی و فعال/غیرفعال به ازای هر محصول
* **بازپرداخت** — کامل و جزئی از طریق API درگاه پرداخت
* **مبادله کالا** — همان محصول یا محصول دیگر
* **اعلان‌های ایمیل** — قالب‌های قابل تنظیم
* **پیگیری دلایل** — تحلیل داده‌های مرجوعی

== Installation ==

1. پوشه wms-explorer را در /wp-content/plugins/ آپلود کنید
2. افزونه را از بخش افزونه‌ها فعال کنید
3. ووکامرس باید نصب و فعال باشد
4. تنظیمات را از منوی مدیریت مرجوعی > تنظیمات انجام دهید
